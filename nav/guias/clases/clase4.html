Decimal a Romano

Creamos un proyecto nuevo en java File..New Project...Java/Java Application

En este momento deberias tener una clase main vacia.

Procedemos a crea una nueva clase dentro del mismo paquete, el paquete es el que contiene la clase principal que se crea por defecto, asegurate de crearla allí mismo.

Ya creada tu clase que en nuestro caso la llamamo Romano empezaremos creando los vectores necesarios para la aplicacion de la conversion.

El primer vector hace referencia a las unidades comunes de los numeros, en este caso del 0-9, recordemos que los vectores almacenan posiciones y en cada posicion tiene un dato. Los demas vectores hacen referencia a los valores de decenas 10-100 Decenas, 100-1000 Centenas, 1000-9999 Unidades de mil.

El metodo convertirRomanos, que hace parte de la clase Romano.java, se le pasa un parametro de tipo entero, en este caso es el numero que se quiere convertir a romano, primero se verifica si el numero pertenece a las decenas, centenas o unidades de mil, y dependiendo del condicional, este obtiene el valor del resultado. Pero miremos mas alla de este simple metodo y veamos que hace en realidad que se obtenga el resultado deseado.

El metodo convertir unidad unicamente, devuelve el valor de la posicion en la cual se encuentra el numero que se esta pasando, por ejemplo, si se envio el numero 4, entonces busca en el vector de unidad la posicion 4, en este caso 4-1=3 y devuelve el valor de esa posicion.

El metodo convertirDecena, primero obtiene el modulo del numero, por ejemplo si es 12, el modulo seria 2, luego se le resta ese modulo al numero para que al final de un numero exacto, en este ejemplo 10, 12-2=10. Luego de haber obtenido el segundo numero obtenemos entonces el primer numero de la posicion en el vector, como es 10 entonces la division de 10/10 = 1 y ese uno se le asigna al vector, y obtiene el valor de X, y el primer numero encontrado en el caso de la variable se llama segundo, se manda al vector de la unidad para que obtenga su valor, en este caso II, para asi concluir con el resultado de XII.

El metodo convertirCentena realiza el mismo proceso anterior, solo que en este caso en vez de llamar a los vectores, simplemente se llama al vector de la centena luego de haber obtenido los valores del modulo y el primero numero, y se envia el resto del numero al metodo anterior.

De esta manera, obtenemos el modulo del numero entre 100, luego le restamos ese modulo para que de un numero exacto, ejemplo 125-25 = 100 y luego obtenemos el valor inicial (1)para este ejemplo, se obtiene del vector el valor de 1 y se envia el 25 al metodo de convertir decena.

El metodo final obtiene realiza el mismo proceso que los metodos anteriores, en este caso divide los valores por 1000 porque son unidades de mil.
